# LivroAberto

Sistema de avaliação de leitura por chat com IA para projeto literário escolar.

## Objetivo
Avaliar evidências de leitura e compreensão de livros por meio de uma conversa guiada, com relatório para revisão docente.

## Escopo do MVP
- Cadastro de livros e turmas
- Upload/indexação de base textual da obra
- Sessão de chat avaliativo com perguntas adaptativas
- Scoring e parecer com nível de confiança
- Painel do professor com histórico e revisão
