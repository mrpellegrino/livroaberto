# LivroAberto Frontend

Frontend React + Vite do produto para escolas.

## Telas implementadas
- Landing page do produto
- Login
- Dashboard
- Escolas
- Turmas
- Livros
- Atividades (com ajuste de pesos da rubrica)
- Avaliacao (chat)
- Resultados

## Configuracao
Crie `.env`:

```bash
VITE_API_URL=http://localhost:3000
```

## Executar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
